#  Personal Portfolio Website

A personal portfolio website built with vanilla HTML, CSS, and JavaScript as part of **Task-04** at **Prodigy Infotech** internship.

---

## 📌 Project Overview

This is a fully responsive, dark-themed personal portfolio website that showcases skills, projects, education, and contact information. Designed with an editorial aesthetic using Playfair Display and DM Mono typefaces.

---

## 🗂️ File Structure

```
portfolio/
├── index.html      # Main HTML structure & content
├── style.css       # All styling, theme, animations, responsive rules
├── script.js       # Scroll reveal & active nav highlight logic
└── README.md       # Project documentation
```

## 🛠️ Technologies Used

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (Custom Properties, Grid, Flexbox, Animations) |
| Behaviour | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts — Playfair Display, DM Mono, DM Sans |


---

## ✨ Features

- **Dark theme** with deep navy-black backgrounds
- **Sticky navigation** with frosted glass backdrop blur
- **Scroll reveal animations** using IntersectionObserver
- **Active nav highlight** on scroll
- **Responsive layout** — works on desktop, tablet, and mobile
- **Hero section** with staggered fade-up animations
- **Skills section** with color-coded pill badges per category
- **Project rows** with hover border and arrow animation
- **Contact section** with slide-in link hover effects

## 🚀 How to Run

1. Download or clone the repository
2. Make sure all 3 files are in the **same folder**:
   ```
   index.html
   style.css
   script.js
   ```
3. Open `index.html` in any modern browser
